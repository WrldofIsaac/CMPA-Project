# Setup Notes

## Step 1: Development Environment
- Use VS Code as the code editor.
- Install the Live Server extension for local preview.
- Use a web browser like Chrome to test the site.

## Step 2: GitHub Repository
Suggested repository name: `worship-night-project`

Files included:
- `README.md`
- `index.html`

## Step 3: Verify Your Pipeline
1. Open the project folder in VS Code.
2. Make a small change to `index.html` or `README.md`.
3. Save the file.
4. Run:
   - `git init`
   - `git add .`
   - `git commit -m "Initial project setup"`
   - `git branch -M main`
   - `git remote add origin YOUR_GITHUB_REPO_URL`
   - `git push -u origin main`
5. Refresh GitHub and confirm the files appear.
